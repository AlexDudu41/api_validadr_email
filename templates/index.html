Bem-vindo ao Verificador de E-mails

Como Utilizar a API

  1. Verificar a validade de um e-mail:

    Faça uma requisição GET para o endpoint /verificar/<email>, substituindo <email> pelo e-mail que você deseja verificar.

    Exemplo: GET /verificar/teste@example.com

    O retorno será um JSON informando se o e-mail é válido e, se não for, sugestões de domínios.

  2. Verificar se um e-mail está ativo:

    Faça uma requisição GET para o endpoint /verificar_atividade/<email>, substituindo <email> pelo e-mail que você deseja verificar.

    Exemplo: GET /verificar_atividade/teste@example.com

    O retorno será um JSON informando se o e-mail parece estar ativo.
